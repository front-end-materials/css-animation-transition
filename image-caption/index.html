Simple caption-on-hover animation

The figcaption holds any amount of text and appears on hover. To ensure legibility, especially if the image is high-contrast, the caption uses backdrop-filter: blur(4px).

In the example on the right, CSS line-clamp is set to 5 to create an ellipsis when there's more than 5 lines. You need to adjust the line-height to make sure the top of the sixth hidden line doesn't show.

Generally it's good to avoid any -webkit prefixes and wait for proper browser support, but it was too tempting…
